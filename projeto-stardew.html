Sistema meteorológico com armazenamento em MongoDB e backend estruturado para manipulação e análise de dados -TechnoFarming

Sistema desenvolvido para coleta, processamento e visualização de dados meteorológicos em tempo real, utilizando integração com MongoDB e exibição dinâmica em gráficos. O projeto permanece online, com a interface funcional e estrutura completa, porém sem ingestão ativa de dados no momento.

Problema

Necessidade de monitorar dados climáticos em tempo real e disponibilizar essas informações de forma visual e acessível.

Solução

Desenvolvimento de um sistema integrado com sensores meteorológicos, enviando dados para o MongoDB e exibindo as informações em um dashboard web com gráficos dinâmicos.

Tecnologias

  • HTML
  • CSS
  • JavaScript
  • MongoDB
  • Vercel
  • APIs - Integração de dados
  • Gráficos - Chart.js

Resultado

- Atualização de dados em tempo real
- Integração com banco de dados na nuvem
- Visualização com gráficos dinâmicos
- Arquitetura cliente-servidor
